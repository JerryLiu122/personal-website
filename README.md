# Personal Website

A full-stack personal website project built with React, TypeScript, Tailwind CSS, Spring Boot, PostgreSQL, Docker, AWS, and GitHub Actions.

## Tech Stack

### Frontend
- React
- TypeScript
- Tailwind CSS
- Vite

### Backend
- Spring Boot
- Spring Security
- Spring Data JPA
- PostgreSQL

### DevOps / Deployment
- Docker
- GitHub
- GitHub Actions
- AWS Amplify Hosting
- AWS Lightsail Container
- Amazon RDS PostgreSQL

## Project Structure

```text
personal-website/
├── frontend/       # React frontend
├── backend/        # Spring Boot backend
├── database/       # SQL schema files
└── README.md
````

## Local Development

### Frontend

```bash
cd frontend
npm install
npm run dev
```

### Backend

```bash
cd backend
./mvnw spring-boot:run
```
